## Description
Nest template.
Build with [Nest](https://github.com/nestjs/nest) framework TypeScript.
Uses swagger for documentation.

## Mongodb branch
This branch is meant to configure the Mongodb database.


## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
